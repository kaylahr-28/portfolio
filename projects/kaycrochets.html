This website reflects proper usage of responsive design, utilizing HTML and CSS to reformat layouts and resize elements appropriately for various browser sizes. Completed in IGME 235: Introduction to Web Technology for Game Developers.

The Process

For this solo class project, I decided to make my website focused on one of my favorite hobbies: crocheting. After completing Phase 1 of the project, where we were tasked to digitally recreate a layout from a printed media source, we then moved into Phase 2, in which we redesigned our digital layout to display our own content. I wanted to keep the magazine feel of the original, so I continued to use the column look from Phase 1. I chose a playful color palette and fonts to reflect the feeling I wanted to invoke within the user. Throughout the process of completing this project, I learned how to effectively use CSS flexboxes and grids to develop website page layouts.
